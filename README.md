# My Resume Website

This is my personal resume website. It’s a simple project where I share my skills, education, and projects in one place.  
The goal is to have a clean and easy-to-read portfolio that I can share with recruiters, friends, and collaborators.

---

## Features
- Shows my resume in a website format
- Sections for **About Me**, **Skills**, **Education**, **Projects**, and **Contact**
- Responsive design (works on mobile and desktop)

---

## Tech Used
- HTML, CSS, JavaScript  
- (Optional) React.js for interactive components  
- Hosted on GitHub Pages / Netlify / Vercel
